---
title: "Blacklist API"
api: "GET https://api-c.walletlabels.xyz/blacklist/label"
description: "Retrieve a curated list of blacklisted addresses from authoritative sources. This dataset includes addresses from USDC and USDT blacklists, OFAC sanctioned addresses, EtherScamDB confirmed phishing entities, and Uniswap banned addresses (TRM Labs)."
---

### Authorization

<ResponseField
  name="x-api-key"
  type="string"
  default="demo-api-key-x-00000"
  initialValue="demo-api-key-x-00000"
  required
>
  Your API key for authentication.
</ResponseField>

### Body

<ParamField query="address" type="string" required>
  Specify up to 10 Solana addresses for which you want to retrieve labels, separated
  by commas.
</ParamField>
<ParamField query="limit" type="string">
  Specify number of shown results.
</ParamField>
<ParamField query="offset" type="string">
  Specify the starting point in the list of results for pagination. e.g.
  `offset=0`, `offset=1`, etc..
</ParamField>

### Response

<ResponseField name="data" type="array">
  An array that encapsulates the labels corresponding to the Solana address provided.

<Expandable title="Label Object">

<ResponseField name="address_name" type="string">
  The designated name associated with the specific Solana address.
</ResponseField>

{" "}

<ResponseField name="address" type="string">
  The Solana address for the label.
</ResponseField>

{" "}

<ResponseField name="label_type" type="string">
  Categories like 'nft', 'gaming', etc., to define the nature of the label.
</ResponseField>

{" "}

<ResponseField name="label_subtype" type="string">
  Additional attributes, like nf_token_contract or aggregator_contract, for a
  more granular understanding of the label.
</ResponseField>

{" "}

<ResponseField name="label" type="string">
  The general name or title of the label.
</ResponseField>

  </Expandable>
</ResponseField>
