---
title: "Blacklist API"
api: "GET https://api-c.walletlabels.xyz/blacklist/label"
description: "Retrieve a curated list of blacklisted addresses from authoritative sources. This dataset includes addresses from USDC and USDT blacklists, OFAC sanctioned addresses, EtherScamDB confirmed phishing entities, and Uniswap banned addresses (TRM Labs)."
---

### Authorization

<ResponseField
  name="x-api-key"
  type="string"
  default="demo-api-key-x-00000"
  initialValue="demo-api-key-x-00000"
  required
>
  Your API key for authentication.
</ResponseField>

### Body

<ParamField query="address" type="string" required>
  Specify up to 10 addresses for which you want to check if they are
  blacklisted, separated by commas.
</ParamField>
<ParamField query="limit" type="string">
  Specify number of shown results.
</ParamField>
<ParamField query="offset" type="string">
  Specify the starting point in the list of results for pagination. e.g.
  `offset=0`, `offset=1`, etc..
</ParamField>

### Response


<ResponseField name="data" type="array">
An array that encapsulates the blacklist information corresponding to the Ethereum addresses provided.

<Expandable title="Blacklist Object"> 

<ResponseField name="address" type="string"> 
  The Ethereum address that is blacklisted. 
</ResponseField>

{" "}

<ResponseField name="blockchain" type="string">
  The blockchain on which the address is blacklisted.
</ResponseField>

{" "}

<ResponseField name="source_of_blacklisting" type="string">
  The source from which the address was blacklisted.
</ResponseField>

{" "}

<ResponseField name="reason_for_blacklisting" type="string">
  The reason for which the address was blacklisted.
</ResponseField>

{" "}

<ResponseField name="date_of_blacklisting" type="string">
  The date when the address was blacklisted.

</Expandable>
